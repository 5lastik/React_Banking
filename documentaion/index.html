1. Folder structure

  • assets: Contains static files such as images, icons, and fonts.
  • components: Contains reusable UI components.
  • config: Contains configuration files.
  • constants: Contains constant values used throughout the application.
  • scss: Contains SCSS files for styling the application.
  • custom: Contains custom utilities and helpers.
  • hooks: Contains custom React hooks.
  • items: Contains specific items or modules related to the application.
  • navigation: Contains navigation components and configurations.
  • product: Contains files related to product management.
  • screens: Contains screen components for different views or pages.
  • store: Contains Redux store configuration and slices.
  • types: Contains TypeScript type definitions.
  • utils: Contains utility functions and helpers.

2. Install environment

Your computer should have Node.js and npm installed to manage the dependencies. You can download Node.js (which includes npm) from the following link: Node.js

3. Install dependencies

To install the project dependencies, run the following command in the project directory: npm install

4. Run the project

To run the project, use the following command: npm start

5. Build the project

To build the project, run the following command: npm run build

6. Deploy the project

After running the build command, the build folder will be ready for deployment. Move all the files from the build folder to your server.
